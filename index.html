IMPROVING OPEN JUSTICE VISUALIZATIONS: A TUTORIAL

by MINJUST

Introdução

Este site pretende ser um guia de como melhor usar dados disponíveis na plataforma de justiça aberta.

Irão ser mostrados alguns exemplos de como criar visualizações de dados (gráficos) através de alguma tecnologias open-source em formato aberto (XML, CSV JSON).

Visualizações Simples

1. Criar uma visualização com recurso a uma library javascript (chartJS) e dados em CSV

2. Criar uma visualização com recurso a uma library javascript (chartJS) e dados vindos da API do CKAN (JSON)

3. Criar uma visualização no CKAN

4. Criar um gráfico no google maps usando a API do google+api do CKAN
